# DevOps Multi-Tier Application

## 🚀 Project Overview
This project demonstrates a complete DevOps pipeline with:
- Frontend (HTML)
- Backend (Node.js)
- Database (MongoDB)
- Docker & Docker Compose

## 🛠 Tools Used
- Docker
- AWS (optional)
- GitHub

## ▶️ How to Run
docker-compose up --build

## 🌐 Output
Frontend: localhost:3000
Backend: localhost:5000
